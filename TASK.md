AssetFork

Packages\com.nyoro_wrl.assetfork

Unityプロジェクト内で、2つのディレクトリのアセットを同期するライブラリ。
ただし、metaファイルは同期しない。
主に同じアセットを、別のインポート設定で使えるようにするために同期するのがこのライブラリの主目的。

例として、同期元をA、同期先をBとする。
A内に新しくアセットができた場合、またはAにあるアセットがBにない場合は、それをBにコピーする。
A内の既存アセットが更新された場合、またはAのアセットとBのアセットの内容が異なっている場合は、Bにコピーする。
A内の既存アセットが削除された場合は、Bから削除する。
例としていくつかの動作を挙げたが、とにかくA→Bで内容を同期させること。
パフォーマンスを重視すること。

設定は、Editorウィンドウで行う。

同期の設定は、複数作成できるようにする。
同期元と同期先のディレクトリの指定は、それぞれObjectで指定する。

同期のフィルタリングについては、条件を追加していく方式で定義する。
条件が空の場合は全てのアセットを同期する。
条件同士はANDで判定する。

- 条件
  - bool 条件を反転する
  - bool 複数の型を指定可能にする（トグル式）
  - Type ここで指定されたアセットを同期する（反転した場合は、同期しない）
  - List<Type> ここで指定されたアセットを同期する（反転した場合は、同期しない）
